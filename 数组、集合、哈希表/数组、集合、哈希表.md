### 1. 两数之和

给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值** *`target`* 的那 **两个** 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。

你可以按任意顺序返回答案。

#### 方法一：哈希表

- 用map存放遍历过的数字，用数字作为key，下标作为value
- 每遍历一个元素：计算targrt减去当前的值
- 判断这个值是否已经存在于map中
  - 存在：[这个值的下标，当前遍历元素的下标]
  - 不存在：存入map中

```js
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var twoSum = function(nums, target) {
    let map = new Map();
    for(let i=0;i<nums.length;i++){
        const sub = target-nums[i];
        if(map.has(sub)) {
            return [map.get(sub), i];
        }
        map.set(nums[i], i);
    }
};
```

### 153. 寻找旋转排序数组中的最小值

已知一个长度为 `n` 的数组，预先按照升序排列，经由 `1` 到 `n` 次 **旋转** 后，得到输入数组。例如，原数组 `nums = [0,1,2,4,5,6,7]` 在变化后可能得到：

- 若旋转 `4` 次，则可以得到 `[4,5,6,7,0,1,2]`
- 若旋转 `7` 次，则可以得到 `[0,1,2,4,5,6,7]`

注意，数组 `[a[0], a[1], a[2], ..., a[n-1]]` **旋转一次** 的结果为数组 `[a[n-1], a[0], a[1], a[2], ..., a[n-2]]` 。

给你一个元素值 **互不相同** 的数组 `nums` ，它原来是一个升序排列的数组，并按上述情形进行了多次旋转。请你找出并返回数组中的 **最小元素** 。

你必须设计一个时间复杂度为 `O(log n)` 的算法解决此问题。

**示例 1：**

```
输入：nums = [3,4,5,1,2]
输出：1
解释：原数组为 [1,2,3,4,5] ，旋转 3 次得到输入数组。
```

**示例 2：**

```
输入：nums = [4,5,6,7,0,1,2]
输出：0
解释：原数组为 [0,1,2,4,5,6,7] ，旋转 4 次得到输入数组。
```

**示例 3：**

```
输入：nums = [11,13,15,17]
输出：11
解释：原数组为 [11,13,15,17] ，旋转 4 次得到输入数组。
```

#### 方法一：二分查找

- 初始化左右指针
- 比较中间元素的值与right指针对应的值
  - `nums[mid] < nums[right]`：mid到right是升序的，最小值在left和mid中间
  - `nums[mid] > nums[right]`：mid到right之间有旋转点
  - 不会出现`nums[mid] = nums[right]`的情况，循环保证了left < right

```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var findMin = function(nums) {
    let left = 0, right = nums.length - 1;
    while (left < right) {
        const mid = Math.floor((left + right) / 2);
        if (nums[mid] < nums[right]) {
            //说明mid到right是升序的，最小值在left和mid中间
            right = mid;
        } else {
            //mid到right之间有旋转点
            left = mid + 1;
        }
    }
    return nums[left];
};
```

### 33. 搜索旋转排序数组

整数数组 `nums` 按升序排列，数组中的值 **互不相同** 。

在传递给函数之前，`nums` 在预先未知的某个下标 `k`（`0 <= k < nums.length`）上进行了 **向左旋转**，使数组变为 `[nums[k], nums[k+1], ..., nums[n-1], nums[0], nums[1], ..., nums[k-1]]`（下标 **从 0 开始** 计数）。例如， `[0,1,2,4,5,6,7]` 下标 `3` 上向左旋转后可能变为 `[4,5,6,7,0,1,2]` 。

给你 **旋转后** 的数组 `nums` 和一个整数 `target` ，如果 `nums` 中存在这个目标值 `target` ，则返回它的下标，否则返回 `-1` 。

你必须设计一个时间复杂度为 `O(log n)` 的算法解决此问题。

**示例 1：**

```
输入：nums = [4,5,6,7,0,1,2], target = 0
输出：4
```

**示例 2：**

```
输入：nums = [4,5,6,7,0,1,2], target = 3
输出：-1
```

**示例 3：**

```
输入：nums = [1], target = 0
输出：-1
```

####  方法一：二分查找

- 将数组从中间分开后，左右两侧一定有一边是有序的
- 如果mid对应的值等于target，直接返回
- 否则，以mid为分割点分出两个部分，并查看哪部分是有序的
  - 若左侧有序，则判断target在不在左侧的值的范围中
    - 若存在，移动right指针
    - 否则，移动left指针
  - 若右侧有序，则判断target在不在右侧的值的范围中
    - 若存在，移动left指针
    - 否则，移动right指针

```js
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number}
 */
var search = function(nums, target) {
    let left=0,right=nums.length-1;
    while(left<=right){
        let mid=Math.floor((left+right) / 2)

        if(nums[mid]===target) return mid
        if(nums[mid]>=nums[left]){
            //mid和mid左边是有序的
            if(target>=nums[left] && target<nums[mid]) {
                right=mid-1
            }else left=mid+1
        }else{
            //mid右边是有序的
            if(target>nums[mid] && target<=nums[right]) {
                left=mid+1
            }else right=mid-1
        }
    }
    return -1
};
```

**注意：**不允许使用任何内置指数函数和算符，例如 `pow(x, 0.5)` 或者 `x ** 0.5` 。

**示例 1：**

```

**示例 2：**

```
输入：x = 8
输出：2
解释：8 的算术平方根是 2.82842..., 由于返回类型是整数，小数部分将被舍去。
```

#### 方法一： 二分查找

- 计算中间点的平方值
  - 平方值=x：直接返回
  - 平方值<x：
    - 将值赋给result（若没有在整数中查找到，返回的是平方根的整数部分）
    - 在mid值的右部分继续寻找
  - 平方值>x：在mid值的左部分继续寻找

```js
/**
 * @param {number} x
 * @return {number}
 */
var mySqrt = function(x) {
    let left = 0,right = x;
    let result;
    while(left<=right){
        let mid=Math.floor(((left+right) / 2))
        let square = mid*mid
        if(square===x) return mid
        else if(square<x) {
            result=mid;
            left=mid+1
        }else{
            right=mid-1
        }
    }
    return result
};
```

给你一个按照非递减顺序排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的开始位置和结束位置。

```

**示例 3：**

```

输出：[-1,-1]
```

#### 方法一：二分查找

- 将问题拆解为求
  - 数组中数为target的最小索引
    - 若nums[mid]>=target，说明mid和mid左侧可能有target，mid可能是数为target的最小索引，于是在左部继续进行二分查找
  - 数组中第一个大于target的数的索引-1
    - 若nums[mid]>target，说明此时target可能在左部，mid可能是第一个大于target的数的索引，于是在左部继续进行二分查找

```js
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */

var binarySearch=(nums,target,lower)=> {
    //lower为true：查找target对应的最小下标
    //lower为false，查找第一个大于target的下标
    let left=0,right=nums.length-1;
    let ans
    while(left<=right){
        let mid=Math.floor((left+right)/2);
        if(nums[mid]>target || lower && nums[mid]>=target) {
            right=mid-1;
            ans=mid;
        }else{
            left=mid+1;
        }
    }
    return ans;
}
var searchRange = function(nums, target) {
    let ans=[-1,-1];
    let leftIndex=binarySearch(nums,target,true);
    let rightIndex=binarySearch(nums,target,false)-1;
    if(leftIndex<=rightIndex && rightIndex < nums.length && nums[leftIndex] === target && nums[rightIndex] === target){
        ans=[leftIndex,rightIndex];
    }
    return ans;
};
```

### 1011.  在 D 天内送达包裹的能力

传送带上的包裹必须在 `days` 天内从一个港口运送到另一个港口。

传送带上的第 `i` 个包裹的重量为 `weights[i]`。每一天，我们都会按给出重量（`weights`）的顺序往传送带上装载包裹。我们装载的重量不会超过船的最大运载重量。

第 3 天：8

请注意，货物必须按照给定的顺序装运，因此使用载重能力为 14 的船舶并将包装分成 (2, 3, 4, 5), (1, 6, 7), (8), (9), (10) 是不允许的。

**示例 2：**

```
输入：weights = [3,2,2,4,1,4], days = 3
输出：6
解释：
船舶最低载重 6 就能够在 3 天内送达所有包裹，如下所示：

第 2 天：2, 4

**示例 3：**

```typescript
输入：weights = [1,2,3,1,1], days = 4
输出：3
解释：
第 1 天：1
第 2 天：2
第 3 天：3
第 4 天：1, 1
```

#### 方法一：二分查找+贪心

- 设定最低运载能力的范围
  - 左：每天只是运输一件快递，左边界要大于weights中最大的值
  - 右：设所有包裹能在一天内运完，左边界为所有包裹重量的总和
- 使用二分查找的方法，选择一个mid值作为当前的运载能力
  - 计数所需要的天数
    - 使用一个need值判断已使用天数
    - 使用一个cur值判断今天已运输的重量
- 判断need与days的关系
  - need<=days：说明运输能力足够或者偏大，可以选择小一点的运输能力，将范围缩小至mid左部
  - need>days：说明运输能力足够或者偏小，需要选择大一点的运输能力，将范围缩小至mid右部

```js
/**
 * @param {number[]} weights
 * @param {number} days
 * @return {number}
 */
var shipWithinDays = function(weights, days) {
    let left=Math.max(...weights);
    let right=weights.reduce((a,b)=>a+b);
    while(left<right){
        let mid=Math.floor((left+right)/2);
        let need=1, cur=0;
        
        for(const weight of weights){
            if(cur+weight > mid){
                need++;
                cur=0;
            }
            cur+=weight;
        }

        if(need<=days){
            right=mid; //mid可以运输完，选择更小的运载能力
        }else{
            left=mid+1; //mid运输不完，从mid右部继续寻找
        }
    }
    return left;
};
```

/**
 * @param {number[]} candidates
 * @param {number} target
 * @return {number[][]}
 */
var combinationSum = function(candidates, target) {
    let res=[];
    const dfs=(start,temp,sum)=>{
        if(sum>=target){ //剪枝
            if(sum===target) res.push(temp.slice());
            return;
        }
        for(let i=start;i<candidates.length;i++){ //i=start表示可以重复选择
            temp.push(candidates[i]);
            dfs(i,temp,sum+candidates[i]);
            temp.pop();
        }
    }
    dfs(0,[],0); //start=0访问第一个元素
    return res;
};
```

- 对每一行进行操作，假设在第一行，再对第一行的每一列进行操作，检查是否可以放入
  - 可以：继续进行下一行的操作
    - 如果都放满了，存储结果
    - 如果下一行一个都放不了，撤销放入的操作
  - 不可以：直接跳过，对下一列进行操作

```js
/**
 * @param {number} n
 * @return {string[][]}
 */
var solveNQueens = function(n) {
    let res = [];

    const isValid=(row,col,board)=>{
        //检查行
        for(let i=0;i<row;i++){
            if(board[i][col]==='Q') return false;
        }
        //检查左上对角线
        for(let i=row-1,j=col-1;i>=0&&j>=0;i--,j--){
            if(board[i][j]==='Q') return false;
        }
        //检查右上对角线
        for(let i=row-1,j=col+1;i>=0&&j<n;i--,j++){
            if(board[i][j]==='Q') return false;
        }
        return true;
    }
    
    const fun=(row,board)=>{
        //都已经放满了
        if(row===n){
            res.push(board.map((item)=>item.join("")));
            return;
        }
        //对每一列进行操作
        for(let col=0;col<n;col++){
            if(isValid(row,col,board)){
                board[row][col]='Q';
                fun(row+1,board);
                board[row][col]='.';
            }
        }
    }
    
    const board=Array(n).fill().map(()=>Array(n).fill("."));
    fun(0,board);
    return res;
};
```

#### 方法一：回溯

- 处理完一行再处理下一行
  - 如果j=9，说明这一行处理结束，进行下一行的处理，时i+1，j=0
  - 如果i=9，说明操作结束，直接返回
  - 如果已存在数字，跳过这个格子，j+1

/**
 * @param {character[][]} board
 * @return {void} Do not return anything, modify board in-place instead.
 */
var solveSudoku = function(board) {
    const hasConflict=(r,c,val)=>{
        for(let i=0;i<9;i++){
            if(board[i][c]==val || board[r][i]==val) return true;
        }
        let rowStart=Math.floor(r/3)*3;
        let colStart=Math.floor(c/3)*3;
        for(let i=0;i<3;i++){
            for(let j=0;j<3;j++){
                if(board[rowStart+i][colStart+j] == val) return true;
            }
        }
        return false;
    }
    const fun=(i,j)=>{
        if(j==9){
            i++;
            j=0;
            if(i==9) return true;
        }
        if(board[i][j]!=='.') return fun(i,j+1);
        for(let num=1;num<=9;num++){
            if(hasConflict(i,j,num)) continue;
            board[i][j]=String(num);
            if(fun(i,j+1)) return true;
            board[i][j]='.';
        }
        return false;
    }
    fun(0,0);
    return board;
};

### 131. 分割回文串

给你一个字符串 `s`，请你将 `s` 分割成一些 子串，使每个子串都是 **回文串** 。返回 `s` 所有可能的分割方案。

**示例 1：**

```
输入：s = "aab"
输出：[["a","a","b"],["aa","b"]]
```

**示例 2：**

```
输入：s = "a"
输出：[["a"]]
```

#### 方法一：回溯

```js
/**
 * @param {string} s
 * @return {string[][]}
 */
var partition = function(s) {
    let res = [];
    
    // 正确的回文判断函数
    const isPalindrome = (str, left, right) => {
        while (left < right) {
            if (str[left] !== str[right]) return false;
            left++;
            right--;
        }
        return true;
    }
    
    const backtrack = (start, path) => {
        // 如果已经分割到字符串末尾，将路径加入结果
        if (start === s.length) {
            res.push([...path]);
            return;
        }
        
        // 尝试所有可能的分割位置
        for (let i = start; i < s.length; i++) {
            // 检查从start到i的子串是否是回文
            if (isPalindrome(s, start, i)) {
                // 如果是回文，加入路径并继续递归
                path.push(s.substring(start, i + 1));
                backtrack(i + 1, path);
                path.pop(); // 回溯
            }
        }
    }
    
    backtrack(0, []);
    return res;
};
```
